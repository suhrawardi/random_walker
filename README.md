# Random Walker


## Howto

It's written in Rust, so install [Rust](https://www.rust-lang.org/tools/install) first:

    $ curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

And then run:

    $ cargo run --release

Or if you have ImageMagick installed, run it with:

    $ ./run.sh
